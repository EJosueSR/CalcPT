Bienvenido a la Calculadora Lógica

Esta herramienta te permite evaluar proposiciones lógicas complejas y generar tablas de verdad.

Instrucciones de uso:

  • Ingresa tu proposición lógica en el campo de texto.
  • Puedes usar variables como p, q, r para generar una tabla de verdad.
  • Utiliza los siguientes operadores:
    • && para AND (Y)
    • || para OR (O)
    • ! para NOT (NO)
  • Usa paréntesis () para agrupar operaciones.
  • También puedes usar valores constantes: true/false o 1/0.

Ejemplo: (p && q) || !r
